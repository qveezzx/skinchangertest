Why Direct Injection Doesn't Work:

  Game Memory
    ↓
    ├─ Application Code (Changeable)
    ├─ Game DLLs (Changeable)
    ├─ Custom Overlays (Possible)
    │
    └─ Steam Overlay (PROTECTED!)
         ↓
         ├─ Encrypted communications
         ├─ Code signing verification
         ├─ Anti-tampering checks

  1. Security - Prevents injection attacks

  4. Licensing - Valve controls access
