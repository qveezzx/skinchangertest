## The Root Causes

1. **PROCESSENTRY32.szExeFile is WCHAR, not char**
   - Must use wide-character functions for comparison
   - Solution: _wcsicmp (wide string compare, case-insensitive) + L"" prefix

2. **SC_GUI doesn't have DrawBorder()**
   - The function simply doesn't exist in this codebase
   - Alternative exists: DrawStrokeRoundedRect() does the same thing
   - Just a different function name/interface

## Why These Fixes Work

| Original | Replacement | Why It Works |
|----------|-------------|--------------|
| `_stricmp` | `_wcsicmp` | WCHAR-safe string comparison |
| `DrawBorder` | `DrawStrokeRoundedRect` | Same visual output, correct API |
| `pe32.szExeFile` | `L"cs2.exe"` | Wide string literal for WCHAR |
| `0.0f` corner radius | Replaces border thickness | Sharp corners = no-radius stroke |
| `8.0f` corner radius | (for beta warning) | Rounded style = matching visual theme |
