❓ COMMON QUESTIONS:

Q: Where's the main UI file?
A: index-pro.html (use this, not index.html)

Q: Do I need to rename it?
A: Optional - you can rename it to index.html to make it default

Q: How do I start the server?
A: node web-server.js (in this directory)

Q: What if port 3000 is busy?
A: Kill the process or edit web-server.js to use different port

Q: Can I add more weapons?
A: Yes! Edit web-server.js weapons array

Q: Can I change colors?
A: Yes! Edit index-pro.html CSS section

Q: Can I make it a single EXE?
A: Yes! See C_PLUS_PLUS_INTEGRATION_GUIDE.md

Q: Will my settings save?
A: Yes! Automatically to skinconfig.json

═════════════════════════════════════════════════════════════════════════════

🔧 TROUBLESHOOTING:

Problem: "node command not found"
Solution: Install Node.js from nodejs.org

Problem: "Port 3000 already in use"
Solution: Change port in web-server.js or kill process

Problem: Browser won't connect
Solution: Check server started, try http://localhost:3000

Problem: Changes not applying
Solution: Check browser console for errors, verify C++ backend running
