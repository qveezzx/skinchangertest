## 🔐 GitHub Authentication

### Using Personal Access Token (Recommended)
1. Get token: https://github.com/settings/tokens
2. Scopes needed: `repo` (full control)
3. When git prompts for password, paste the token
4. (Keep the token in password manager for next time)

### Using SSH (Alternative)
1. Set up SSH key (if not done): `ssh-keygen -t ed25519 -C "email@example.com"`
2. Add to GitHub: https://github.com/settings/keys
3. Git will use it automatically

### Using GitHub CLI (Another Option)
```bash
gh auth login
# Then: git push origin main
```
