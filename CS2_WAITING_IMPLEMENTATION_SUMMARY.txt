🔧 HOW IT WORKS
═════════════════════════════════════════════════════════════════════════════

STATE MACHINE:

  WAITING_FOR_CS2 ←→ LOADING → READY
        ↑                         ↑
        └─────────────────────────┘
            (If CS2 closes)


DETECTION LOGIC:

  Every frame:
    Check if cs2.exe is running
    
    If no cs2.exe + state isn't WAITING
      → Set state to WAITING_FOR_CS2
    
    If cs2.exe running + state is WAITING
      → Set state to LOADING
      → Start 2-second timer
    
    If loading timer > 2 seconds
      → Set state to READY


RENDERING LOGIC:

  If state = WAITING_FOR_CS2
    → Draw waiting screen only
    → Don't draw menu
  
  If state = LOADING
    → Draw loading screen only
    → Don't draw menu
  
  If state = READY
    → Draw normal menu
    → Draw beta warning if active

🧪 TESTING PROCEDURE
═════════════════════════════════════════════════════════════════════════════

Test 1: Waiting State
  • Close CS2 completely
  • Run skinchanger.exe
  • Should see "Waiting for CS2..."
  • App should NOT crash

Test 2: Loading State
  • From waiting state, launch CS2
  • Should see loading bar
  • Bar should fill over 2 seconds
  • Should NOT see menu yet

Test 3: Ready State
  • After loading completes
  • Should see normal menu
  • Menu should be fully functional
  • No loading bar visible

Test 4: Beta Warnings
  • Click on Knives tab
  • Should see warning dialog
  • Click OK
  • Should access knives menu
  • Repeat for Gloves tab

Test 5: CS2 Close
  • From Ready state, close CS2
  • App should return to "Waiting"
  • Should NOT crash
  • Can relaunch CS2
