🚀 TROUBLESHOOTING:

Port 3000 already in use?
   → Kill existing process or change port in web-server.js

Server won't start?
   → Make sure Node.js is installed (node --version)
   → Try: npm install

Browser won't open?
   → Open manually: http://localhost:3000/index-pro.html

Changes not applying?
   → Check browser console for errors
   → Verify skinconfig.json is created
   → Check game process is running
