🎯 WHAT HAPPENS WHEN...
════════════════════════════════════════════════════════════════════════════

CS2 not running:
  → Shows waiting message
  → No menu visible
  → No crashes

CS2 starts:
  → Auto-detected (no user action)
  → Loading bar shows
  → After 2 sec → Menu appears

User presses INSERT:
  → Toggles menu visibility
  → Beta warning stays on top if active

User clicks Knives/Gloves first time:
  → Beta warning shows
  → Blocks menu access
  → Click OK to proceed
  → Warning won't show again that session

CS2 closes:
  → App detects it
  → Returns to waiting state
  → Can relaunch CS2 and continue

📞 SUPPORT / DEBUGGING
════════════════════════════════════════════════════════════════════════════
